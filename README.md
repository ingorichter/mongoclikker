###mongoclikker

[MongoDB](http://mongodb.org) viewer written in [Node](http://nodejs.org). Just clone, run `git submodule update --init` and fire up [mongoclikker](https://github.com/semu/mongoclikker) with 

    [sudo] node mongoclikker.js
